# Power BI Dark Theme (JSON)

This repository contains a custom Power BI report theme created using JSON and tested successfully in Power BI Desktop.

The theme is designed to provide a clean and professional dark layout for Power BI dashboards with consistent colors and improved readability.

---

## Features

- Dark background and light foreground colors  
- Custom data color palette  
- Styled visual titles and text  
- Customized filter pane and outspace pane  
- Improved tooltip appearance  
- Responsive visuals enabled  

---

## Files

- `Power Bi Acterys Repor Theme.json` – Power BI theme file

---

## How to Use

1. Open Power BI Desktop  
2. Go to **View → Themes → Browse for themes**  
3. Select `practice.json`  
4. Apply the theme to your report
